## Description
Glistore is a project to build your own store 100% customizable. This is one of the backend services for glistore.

## Running the app with Docker

```bash
# development
$ docker compose up
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
